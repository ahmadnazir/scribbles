## Build

### Prerequisites

```
sudo apt install python3-pip
pip3 install -U ablog
```

Hack: If the above doesn't work, install it from within the docker container
with the home directory mounted.


### Install

```
source ./scripts/scribbles.sh
```

### Compile

```
scribbles-compile
```

### Cleanup

```
scribbles-clean
```

## Publish

```
scribbles-publish
```
