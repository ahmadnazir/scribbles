GPG¶

Create a new gpg key¶

gpg --default-new-key-algo rsa4096 --gen-key

For me, it looked like:

gpg -K --keyid-format long --with-colons --with-fingerprint

gpg --export -a 1F7BFED782E9E540432E3F48E28E7DA5BE1BAEBF
