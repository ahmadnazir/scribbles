Lists
=====

Numbering
---------

.. code::

   #. Item A
   #. Item B
   #. Item C

*renders to:*

#. Item A
#. Item B
#. Item C

Nested List
-----------

.. code::

   - **Here is a nested list**:
 
     - Item A
     - Item B
     - Item C

*renders to:*

- **Here is a nested list**:

  - Item A
  - Item B
  - Item C
