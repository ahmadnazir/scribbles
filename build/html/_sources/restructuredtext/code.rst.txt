Code Snippets
=============

.. post:: 31/01/2022
   :tags: restructuredtext

Use 3 chars to indent the code snippet.

.. code::

   .. code::

      public static void main()
      {
        System.out.println("Hello World");
      }
 
*renders to:*

.. code::

   public static void main()
   {
     System.out.println("Hello World");
   }

or get the same results with using `::` at the end of the sentence:::

  Some code::

    public static void main()
    {
      System.out.println("Hello World");
    }
