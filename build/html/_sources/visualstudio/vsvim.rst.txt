VsVim
=====

:Version: Visual Studio 2019

`VsVim` is an extension for visual studio for vim keybindings


Chrome tab navigation
---------------------

Use a `.vimrc` file in the `$HOME` directory:

.. code::

   nnoremap <C-PageDown>  :tabnext<CR>
   nnoremap <C-PageUp>    :tabprev<CR>

Reload settings
---------------

`Ctrl-Shift-F12`: https://github.com/VsVim/VsVim/wiki/FAQ#how-can-i-temporarily-disable-vsvim
