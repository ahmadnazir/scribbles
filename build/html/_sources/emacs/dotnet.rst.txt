.NET on Emacs
=============

.. post:: May 20, 2019
   :tags: emacs

Install omnisharp
-----------------


.. code::

   M-x omnisharp-install-server

Enable intellisense
-------------------

.. code::

   fsharp-ac-intellisense-enabled

Auto-complete
-------------

.. code::

   M-x omnisharp-auto-complete
