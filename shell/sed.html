sed¶

Stream editor. Using regular expression, editor the standard input.

Remove the first character¶

echo '_Test' | sed 's/.//'

evaluates to:

Test

Unbuffered output¶

Use the -u switch for unbuffered output.

echo '_Test' | sed -u 's/.//'
